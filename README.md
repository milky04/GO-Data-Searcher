#### このツールについて
「ポケモンGO」の対戦でパーティ編成をする際参考にするためのツールです。  
ポケモンや条件を指定して、そのポケモンと相性補完の良いポケモンを検索します。  
各ポケモンのタイプや、タイプの優秀さを数値化したものを元に算出されます。  
チャットアプリ「Discord」でbotとして使用します。
